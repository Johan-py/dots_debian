### Historia de Usuario 1 – Find

**Como** usuario  
**Quiero** encontrar información dentro del sistema  
**Para** acceder rápidamente a los recursos que necesito.

**Criterios de aceptación**

- El sistema debe tener una barra de búsqueda.
    
- Los resultados deben mostrarse ordenados por relevancia.
    
- El usuario puede filtrar resultados.
    

---

### Historia de Usuario 2 – Work

**Como** usuario  
**Quiero** trabajar en mis tareas dentro del sistema  
**Para** completar mis actividades de forma organizada.

**Criterios de aceptación**

- El usuario puede crear tareas.
    
- El usuario puede editar tareas.
    
- El sistema guarda automáticamente los cambios.
    

---

### Historia de Usuario 3 – Ask for help

**Como** usuario  
**Quiero** solicitar ayuda cuando tengo un problema  
**Para** resolver dificultades mientras uso el sistema.

**Criterios de aceptación**

- El usuario puede enviar una solicitud de ayuda.
    
- El sistema registra la solicitud.
    
- Un administrador puede responder la solicitud.
