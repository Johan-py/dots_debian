conceptos basicos
# escaneo de puertos
es una tecnica usada para desgubrir puertos abiertos en un sistema
permite identificar servicios en ejecucion

## Escaneo activo - pasivo
### Activo
es directo y presiso, su deteccion es sencilla y puede ser mas lento debido a medidas de seguridad del sistema objetivo

### Pasivo
es mas discreto, ofrece una via menos instrusiva, aunque puede no ser tan detallada 

la eleccion depende del objetivo

# Estados de los puertos
- Open: puerto abierto 
- Closed: puerto cerrado, sin servicio
- wrapped

# 3-way handshake
El 3-way handshake es un procedimiento en TCR/IP, para establecer una conexion entre dos dispositivos en una red
- SYN (synchronize)
- SYN-ACK(synchronize-acknowledgm ent)
- ACK (acknowledgment)


# Que es NMAP

Nmap (Network Mapper) es una herramienta que se usa para el descubrimiento de redes.
- descubrir host y servicios en una red enviando paquetes y analizando las respuestas 
- recopilar informacion sobre los sistemas operativos
- descubrir puertos abiertos, servicios ejecutandose y sus versionso
- descubrir vurneravilidades
